В базе данных Redis подберите коллекцию для подсчета посещений с определенных IP-адресов.
Запись IP - количество посещений
127.0.0.1:6379> sadd 123.01.01 1
(integer) 1
127.0.0.1:6379> sadd 123.02.01 2
(integer) 1
127.0.0.1:6379> sadd 123.03.01 3
(integer) 1
127.0.0.1:6379> sadd 123.04.01 4
(integer) 1

При помощи коллекции SDIFF cчитаем количество посещений с определенных IP-адресов:

127.0.0.1:6379> sdiff 123.01.01
1) "1"
127.0.0.1:6379> sdiff 123.03.01
1) "3"

При помощи базы данных Redis решите задачу поиска имени пользователя по электронному адресу и наоборот, поиск электронного адреса пользователя по его имени.
Поиск электронного адреса по имени пользователя:
127.0.0.1:6379> hmset users anna anna@mail.ru maria maria@mail.ru
OK
127.0.0.1:6379> hget users maria
"maria@mail.ru"
поиск имени пользователя по электронному адресу:
127.0.0.1:6379> hmset emails anna@mail.ru anna maria@mail.ru maria elena@mail.ru elena jane@mail.ru jane
OK
127.0.0.1:6379> hget emails jane@mail.ru
"jane"
